# Note on the Repository History

The visible commit history of this repository is intentionally
condensed. The work was developed iteratively over several weeks,
with a substantially larger number of intermediate commits than the
five visible on the public branch. To keep the history focused on
the algorithm and its validation, the development was squashed into
thematic commits when the repository was prepared for public
release.

Discarded algorithmic variants, earlier paper drafts, and
intermediate experimental data are preserved under `archive/`,
`experiments/archive/`, `data/archive/`, and `paper/archive/`. None
of the archived material supports any claim in the current paper;
the explicit mapping from each headline number to the script and
data file that produced it is given in
[`RESULTS_MANIFEST.md`](../RESULTS_MANIFEST.md).

If you want access to the granular development history (individual
commits, decisions made along the way), please contact the
maintainer directly. The full history is available on request.
